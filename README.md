# KulturSmåland

En webbplats för kulturintresserade som vill upptäcka museer, slott och kyrkor i Småland och på Öland.

**Live:** https://gk-lnu.github.io/1me302-projekt/

## Om projektet

KulturSmåland riktar sig till kulturintresserade i Småland och Öland. Användaren kan bläddra bland kulturplatser, filtrera på kategori, kommun och avstånd, sortera resultaten och öppna en detaljvy med mer information och en karta för varje plats.

## Funktioner

- Filtrering på kategori, kommun och avstånd (flera filter samtidigt)
- Sortering av platser
- Sökfunktion
- Detaljvy med bild, pris, adress, betyg, karta och länk till platsens webbplats
- Avståndsberäkning utifrån användarens position
- Responsiv layout för mobil

## Teknik

- HTML, CSS och JavaScript
- Data hämtas från SMAPI (Smålands API)
- Kartor via Leaflet
- Publicerad med GitHub Pages
